### Простая структура (модули в одной папке)

Создайте папку проекта.
Поместите туда `main.py` (точка входа) и, например, `utils.py` (вспомогательный файл).

В `utils.py` определите функции:
```python
# utils.py
def greet(name):
    return f"Hello, {name}!"

```

В `main.py` импортируйте и используйте:

```python
# main.py
from utils import greet

print(greet("World"))
```

>Запуск: ```python main.py``` (из папки проекта).

### Структура пакета (с папками)

Создайте папку `my_project`.
Внутри my_project создайте `__init__.py` (может быть пустым), `main.py` и папку `components`.
В папку components добавьте `__init__.py` и `helper.py`.

helper.py:
```python
# my_project/components/helper.py
def add(a, b):
    return a + b

```

main.py:
```python
# my_project/main.py
from components.helper import add

result = add(5, 3)
print(f"Result: {result}")
```

>Запуск: ```python my_project/main.py``` (из папки выше my_project).

### Запуск как отдельного скрипта
   >Для запуска любого файла используйте `python имя_файла.py` в терминале.

   >Для выполнения файла как программы используйте   
   ```python -m my_project.main (если my_project — пакет)```. 
   
### Продвинутый способ (установка пакета)
   Для больших проектов создайте файл `setup.py`, который описывает ваш проект.
   Это позволяет устанавливать проект в виртуальное окружение с помощью `pip install .`, делая его доступным из любого
   места и управляя зависимостями.
